# Web Application Development: Assignment 2(A)


## 📝 Problem Statement
Create version control account on GitHub and using Git commands to create repository and push 
your code to GitHub.

---

## 🛠️ Git Commands Implemented
Below is the sequence of commands used to fulfill the assignment requirements:


| Action | Command |
| :--- | :--- |
| **Initialize Git** | `git init` |
| **Stage Changes** | `git add .` |
| **Commit Changes** | `git commit -m "Assignment 2A: Initial Setup"` |
| **Rename Branch** | `git branch -M main` |
| **Link Remote** | `git remote add origin <your-repo-url>` |
| **Push to Cloud** | `git push -u origin main` |

---

## 💻 Tech Stack
*   **Version Control:** Git (CLI)
*   **Hosting Service:** GitHub
*   **Editor:** Visual Studio Code
*   **Terminal:** PowerShell / Git Bash
